1. jQuery Libraries

Cómo integrar JQuery a HTML:

Primeramente, necesitamos comprender que un documento JS, se debe ejecutar una vez que el DOM esté construido, para esto tenemos algunas herramientas para asegurarnos que su ejecución suceda en ese momento.

El Document Ready Event: Esto evita que se ejecute código jQuery antes de que el documento haya terminado de cargarse (esté listo). Esto permite agregar la referencia del archivo JS o código directo tanto en el head como en el body.

Sin embargo esto no es necesario si colocas la referencia script al final del body o código directo. (Porque el DOM ya habrá sido creado)

$ (selector).acción( );

Es importante reconocer que dentro de un archivo Js e incluso dentro del document ready mencionado anteriormente, cómo este trabaja, para esto debemos conocer sus partes y sus respectivas funciones:

A) $: Es la forma en que jQuery se llama (es como decir “usa jQuery”)
B) Selector: Es lo que quieres seleccionar, igual que en CSS (#id, .clase, tag)
c) Acción :Es la función o método que quieres ejecutar

En la siguiente pestaña podremos ver que existen diferentes tipos de selectores, siendo bastante similar a CSS.

Selección de elementos

Un selector de elementos en jQuery es una expresión que permite identificar y seleccionar uno o varios elementos del DOM (Document Object Model) para manipularlos mediante métodos jQuery.

Los selectores por etiqueta en jQuery son los más básicos y se usan para seleccionar elementos HTML por su nombre de etiqueta directamente.

Código: Referencia 

// Seleccionar todos los párrafos p
$("p");

// Seleccionar todas las imágenes img
$("img");

// Seleccionar todos los encabezados h1
$("h1");

// Seleccionar todas las listas ul
$("ul");

// Seleccionar todos los botones button
$("button");

Permite identificar un único elemento (o varios si comparten el mismo ID, aunque no es recomendable) para manipularlo con jQuery.

Código: Referencia 


// Seleccionar un elemento con ID "titulo"
$("#titulo");

// Seleccionar un botón con ID "btnEnviar"
$("#btnEnviar");

// Seleccionar un div con ID "contenedor"
$("#contenedor");

Permite identificar uno o varios elementos que comparten la misma clase para aplicarles cambios o eventos.

Código: Referencia 


// Seleccionar todos los elementos con la clase "activo"
$(".activo");

// Seleccionar todos los párrafos con la clase "resaltado"
$("p.resaltado");

// Seleccionar todos los divs con la clase "contenedor"
$("div.contenedor");

los selectores por jerarquía permiten seleccionar elementos en función de su relación dentro del árbol DOM.

Código: Referencia 

// Seleccionar todos los li que están dentro de un ul
$("ul li");

// Seleccionar todos los span dentro de un div
$("div span");

// Seleccionar los p que son hijos directos de un div
$("div > p");

// Seleccionar el primer li dentro de cada ul
$("ul li:first-child");

// Seleccionar el último p dentro de un section
$("section p:last-child):

Los selectores por atributo en jQuery permiten seleccionar elementos HTML basándose en los atributos que tienen. Se escriben entre corchetes []

Código: Referencia 

// Seleccionar todos los elementos que tienen el atributo title
$("[title]");

// Seleccionar todos los inputs con atributo type="text"
$("input[type='text']");

// Seleccionar todos los enlaces con atributo href
$("a[href]");

// Seleccionar imágenes con atributo alt específico
$("img[alt='logo']");

// Seleccionar botones con atributo disabled
$("button[disabled]");

Los selectores combinados por atributo en jQuery permiten mezclar atributos con etiquetas, clases o incluso otros atributos.

Cómo funciona:

  • etiqueta + atributo:
  • atributo + clase
  • varios atributos
Código: Referencia 

// Seleccionar todos los inputs de tipo texto dentro de un formulario
$("form input[type='text']");

// Seleccionar enlaces a con atributo target="_blank"
$("a[target='_blank']");

// Seleccionar imágenes con atributo alt y clase específica
$("img[alt='logo'].img-fluid");

// Seleccionar botones con atributo disabled y clase btn-primary
$("button[disabled].btn-primary");

// Seleccionar div con atributo data-role="admin"
$("div[data-role='admin']");

Acceso y manipulación del DOM

Descripción de lo explicado

La inserción al DOM consiste en añadir nuevos nodos o contenido HTML en posiciones específicas respecto a un elemento seleccionado. jQuery facilita esto mediante métodos simples que evitan manipulación manual del DOM.

Existen 3 Tipos:
  • Inside(dentro)
  • Outside(fuera)
  • Around(Alrededor)

Se dispara cuando el usuario realiza dos clics seguidos sobre un elemento, útil para acciones rápidas o avanzadas.

Código - Referencia de Ejemplo 

$("#cuadroDblClick").dblclick(function () {
  const cuadro = $(this);

  if (cuadro.hasClass("agrandado")) {
    // Volver al tamaño original
    cuadro
      .removeClass("agrandado")
      .css({
        "background-color": "lightblue",
        width: "150px",
        height: "150px"
      })
      .text("Doble clic aquí");
  } else {
    // Agrandar
    cuadro
      .addClass("agrandado")
      .css({
        "background-color": "orange",
        width: "250px",
        height: "250px"
      })
      .text("¡Doble clic detectado!");
  }
});

Mouseover ocurre cuando el puntero entra al área del elemento, y mouseout cuando sale. Son ideales para efectos visuales, menús o resaltar elementos.

Código - Referencia de Ejemplo 

// Evento mouseover: cuando el mouse entra
$("#cuadroHover").mouseover(function () {
  $(this)
    .css("background-color", "#b3e5fc")
    .text("Mouse encima 👀");
});

// Evento mouseout: cuando el mouse sale
$("#cuadroHover").mouseout(function () {
  $(this)
    .css("background-color", "#dcdcdc")
    .text("Pasa el mouse aquí");
});

Se ejecuta cada vez que el usuario libera una tecla dentro de un input, textarea o cualquier campo editable. Útil para validaciones en tiempo real o búsquedas dinámicas.

Código - Referencia de Ejemplo 

// HTML
<input type="text" id="inputKeyup" placeholder="Escribe algo..." class="form-control" style="width: 250px;">
<p id="mensajeKeyup" class="mt-3">Número de caracteres: 0</p>

// jQuery
$("#inputKeyup").keyup(function () {
  const texto = $(this).val();
  $("#mensajeKeyup").text("Número de caracteres: " + texto.length);
});

Este evento se activa cuando un formulario es enviado, ya sea con un botón o la tecla Enter. Permite validar datos antes de que se procese el envío.

Código - Referencia de Ejemplo 

// HTML
<form id="formSubmit" class="d-flex flex-column align-items-center">
  <input type="text" id="nombreSubmit" placeholder="Escribe tu nombre" class="form-control mb-3" style="width: 250px;" required>
  <button type="submit" class="btn btn-primary">Enviar</button>
  <p id="mensajeSubmit" class="mt-3 text-success"></p>
</form>

// jQuery
$("#formSubmit").submit(function (e) {
  e.preventDefault(); // Previene el envío real

  const nombre = $("#nombreSubmit").val();
  if (nombre.trim() === "") {
    $("#mensajeSubmit")
      .text("Por favor, ingresa tu nombre")
      .css("color", "red");
  } else {
    $("#mensajeSubmit")
      .text("¡Formulario enviado correctamente, " + nombre + "!")
      .css("color", "green");

    $(this)[0].reset(); // Opcional: reinicia el formulario
  }
});
