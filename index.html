1. jQuery Libraries

Cómo integrar JQuery a HTML:

Primeramente, necesitamos comprender que un documento JS, se debe ejecutar una vez que el DOM esté construido, para esto tenemos algunas herramientas para asegurarnos que su ejecución suceda en ese momento.

El Document Ready Event: Esto evita que se ejecute código jQuery antes de que el documento haya terminado de cargarse (esté listo). Esto permite agregar la referencia del archivo JS o código directo tanto en el head como en el body.

Sin embargo esto no es necesario si colocas la referencia script al final del body o código directo. (Porque el DOM ya habrá sido creado)

$ (selector).acción( );

Es importante reconocer que dentro de un archivo Js e incluso dentro del document ready mencionado anteriormente, cómo este trabaja, para esto debemos conocer sus partes y sus respectivas funciones:

A) $: Es la forma en que jQuery se llama (es como decir “usa jQuery”)
B) Selector: Es lo que quieres seleccionar, igual que en CSS (#id, .clase, tag)
c) Acción :Es la función o método que quieres ejecutar

En la siguiente pestaña podremos ver que existen diferentes tipos de selectores, siendo bastante similar a CSS.

    <script src="JS/JPropio.js"></script> <!-- Al final del body: NO necesita document.ready -->

      // Código que manipula elementos ya cargados
    </script>

  </body>
</html>

2. Google APIs

Google Maps

El objeto Geolocation de HTML5 permite acceder a la ubicación geográfica del dispositivo. No es parte de la API de Google Maps, pero se usa comúnmente para obtener la posición actual del usuario.

Código: Referencia a Objeto Geolocation 

                          1. Referencia a Objeto Geolocation:
                          if (navigator.geolocation) {
                            console.log("Geolocation está disponible");
                          } else {
                            console.log("Geolocation no está disponible");
                          }

                          2. Obtener la Posición Actual:
                          navigator.geolocation.getCurrentPosition(
                            (position) => {
                              const lat = position.coords.latitude;
                              const lng = position.coords.longitude;
                              console.log(`Latitud: ${lat}, Longitud: ${lng}`);
                            },
                            (error) => {
                              console.error("Error al obtener la posición:", error);
                            }
                          );
Ejemplo Visual - Mapa Geolocation
Resultado: Haz clic en el botón para verificar si Geolocation está disponible

El método getCurrentPosition() acepta un callback para manejar errores, como la denegación de permisos o la indisponibilidad del servicio. Tipos de errores comunes:

PERMISSION_DENIED: El usuario no permitió el acceso a la ubicación.

POSITION_UNAVAILABLE: La ubicación no está disponible.

TIMEOUT: El servicio tardó demasiado en responder.

Código: Manejo de errores 

                            navigator.geolocation.getCurrentPosition(
                              (position) => {
                                const lat = position.coords.latitude;
                                const lng = position.coords.longitude;
                                console.log(`Latitud: ${lat}, Longitud: ${lng}`);
                              },
                              (error) => {
                                switch(error.code) {
                                  case error.PERMISSION_DENIED:
                                    console.error("El usuario denegó la solicitud de geolocalización.");
                                    break;
                                  case error.POSITION_UNAVAILABLE:
                                    console.error("La información de ubicación no está disponible.");
                                    break;
                                  case error.TIMEOUT:
                                    console.error("La solicitud de ubicación ha caducado.");
                                    break;
                                  default:
                                    console.error("Error desconocido:", error);
                                }
                              }
                            );

Ejemplo: Ruta desde la posición actual del usuario hasta el Parque Central de Alajuela
Dirección: Parque Central, Alajuela, Costa Rica
Coordenadas: 10.0162° N, -84.2118° W

Código: Trazar ruta al Parque Central, Alajuela, Costa Rica. 

// Coordenadas del Parque Central de Alajuela
const DEST_PARQUE_CENTRAL = { lat: 10.0162, lng: -84.2118 };

function initRouteMap() {
  const mapDiv = document.getElementById('map');
  if (!mapDiv) return;

  // Crear mapa centrado en el Parque Central
  const map = new google.maps.Map(mapDiv, {
    center: DEST_PARQUE_CENTRAL,
    zoom: 13,
    mapTypeControl: true,
  });

  // Marcador del destino
  new google.maps.Marker({
    position: DEST_PARQUE_CENTRAL,
    map: map,
    title: 'Parque Central de Alajuela',
    icon: 'https://maps.google.com/mapfiles/ms/icons/red-dot.png'
  });

  // Obtener ubicación del usuario y trazar ruta
  if (navigator.geolocation) {
    navigator.geolocation.getCurrentPosition(
      (position) => {
        const userLocation = {
          lat: position.coords.latitude,
          lng: position.coords.longitude
        };

        // Marcador de ubicación del usuario
        new google.maps.Marker({
          position: userLocation,
          map: map,
          title: 'Tu ubicación',
          icon: 'https://maps.google.com/mapfiles/ms/icons/blue-dot.png'
        });

        // Configurar DirectionsService y Renderer
        const directionsService = new google.maps.DirectionsService();
        const directionsRenderer = new google.maps.DirectionsRenderer({
          map: map,
          suppressMarkers: false
        });

        // Solicitar la ruta
        directionsService.route(
          {
            origin: userLocation,
            destination: DEST_PARQUE_CENTRAL,
            travelMode: google.maps.TravelMode.DRIVING
          },
          (response, status) => {
            if (status === google.maps.DirectionsStatus.OK) {
              directionsRenderer.setDirections(response);
              
              // Obtener información de la ruta
              const route = response.routes[0].legs[0];
              console.log(`Distancia: ${route.distance.text}`);
              console.log(`Duración: ${route.duration.text}`);
            } else {
              console.error('Error al trazar la ruta:', status);
            }
          }
        );
      },
      (error) => {
        console.error('Error de geolocalización:', error);
        alert('No se pudo obtener tu ubicación. Por favor, permite el acceso a la ubicación.');
      }
    );
  } else {
    alert('Tu navegador no soporta geolocalización.');
  }
}
